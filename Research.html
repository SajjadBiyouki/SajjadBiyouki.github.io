Data-driven Fuzzy Modeling for Thyroid Function Diagnosis and Nodule Detection based on Radioisotope Images

The thyroid is one of the human glands the diseases of which have spread worldwide. The thyroid gland produces hormones whose main function is to balance the metabolism and cells activities. There are various tools and techniques for diagnosing thyroid diseases; however, radioisotope imaging is the most recent technique which can specify thyroid diseases more accurately. This paper presents a data-driven system for diagnosing the most prevalent diseases of thyroid (i.e. thyroid function disorders and thyroid nodules) based on thyroid radioisotope images. The proposed fuzzy image processing model consists of four distinct phases including pre-processing, segmentation, feature extraction, and classification. In the segmentation, a new modified clustering algorithm, namely Adaptive Possibilistic Fuzzy C-mean (IT2APFCM), is applied to segment an image into similar classes. The PFCM algorithm has drawbacks in defining the coefficients of membership degrees. It is, indeed,an adaptive algorithm that can define these coefficients automatically based on the input data.
